# 🚀 TaskFlow - Fullstack Task Manager


Fullstack task management system with separated frontend and backend architecture.

## 📌 Sobre o projeto

Este projeto tem como objetivo simular um sistema real de gerenciamento de tarefas, permitindo:

- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Excluir tarefas

## 🧠 Tecnologias utilizadas

### Frontend:
- HTML
- CSS
- JavaScript

### Backend:
- Node.js (se estiver usando, me confirma depois)
- API REST
